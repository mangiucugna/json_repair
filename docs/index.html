JSON fixer and linter online

This is a demo page for the Python package json_repair, you can find more information visting the GitHub repository: https://github.com/mangiucugna/json_repair/

How to use: Paste malformed JSON in the left textarea and the repaired JSON will be displayed on the right. This is particularly useful to test LLMs output

Please note: the api is hosted on PythonAnywhere for free. If too many have used it in a day, it might reach quota and not work
